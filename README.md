# EventHub
Academic Event Management System is a Spring Boot–based web application that automates the creation, approval, registration, and management of academic events within a college campus using role-based access.

Objectives
- Reduce manual work in managing academic events
- Provide a centralized platform for event registration and monitoring
- Ensure secure access through role-based authentication
- Improve coordination between students, coordinators, and administrators

User Roles
- Admin – Approves events, manages users, generates reports
- Coordinator – Creates and manages events
- Student – Views events, registers, and submits feedback

Key Features
- User registration and secure authentication
- Role-based dashboards (Admin, Coordinator, Student)
- Event creation, approval, and management workflow
- Online event registration with confirmation
- Feedback and rating system
- User and role management
- Responsive web design

  Technology Stack
- Backend: Spring Boot, Spring MVC, Spring Security
- Frontend: HTML, CSS, Bootstrap
- Database: MySQL
- Tools: Spring Tool Suite (STS)

How to Run the Project
1. Clone the repository
2. Open the project in Spring Tool Suite (STS)
3. Configure MySQL database in `application.properties`
4. Run the application as Spring Boot App
5. Open browser and visit:  http://localhost:8080
